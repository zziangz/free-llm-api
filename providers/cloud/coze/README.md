# Coze - Not Included

**Status:** Excluded from registry

## Reason for Exclusion

Coze is **not included** in this registry due to its **outdated model list**.

While Coze offers free API access, the available models are significantly behind current state-of-the-art offerings from other providers. Including outdated models would not provide value to users who can access more capable, recent models elsewhere.

## What Coze Offers

- Bot building platform by ByteDance
- Free tier available
- OpenAI-compatible API

## Why We Pass

| Issue | Impact |
|-------|--------|
| Outdated models | Latest open-source models (Qwen3, Llama 4, DeepSeek V3) not available |
| Slow updates | Model roster lags months behind other providers |
| Limited selection | Fewer models compared to OpenRouter, Groq, etc. |

## Alternatives

For free access to recent models, consider:

- **[Groq](../groq/)** - Fast inference, latest Llama/Qwen models
- **[OpenRouter](../openrouter/)** - Widest selection of free models
- **[GitHub Models](../github-models/)** - GPT-4o, Llama, Mistral
- **[Ollama Cloud](../ollama-cloud/)** - Large open-source models

## Future Consideration

If Coze updates their model offerings to include recent releases, we may reconsider inclusion. Feel free to open an issue if the situation changes.
